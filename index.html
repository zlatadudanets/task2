1. GET главной страницы (HTTP/1.0)

Здесь мы запрашиваем главную страницу сайта. Метод GET используется для получения данных с сервера. HTTP/1.0 — старая версия протокола.

GET /task2/ HTTP/1.0
Host: u82300.kubsu-dev.ru

Ответ сервера: 200 OK

2. GET внутренней страницы (HTTP/1.1)

Запрашиваем конкретный файл — index.html.Мы проверяем работу новой версии протокола.

GET /task2/index.html HTTP/1.1
Host: u82300.kubsu-dev.ru

Ответ сервера: 200 OK

3. Определение размера file.tar.gz

Используем метод HEAD. HEAD - возвращает только заголовки, без тела файла. Используем для того чтобы узнать размер файла

HEAD /task2/files/file.tar.gz HTTP/1.1
Host: u82300.kubsu-dev.ru

Размер файла: 31 байт

4. Определение медиатипа image.png

Снова используем HEAD и узнаем тип файла

HEAD /task2/files/image.png HTTP/1.1
Host: u82300.kubsu-dev.ru

Content-Type: image/png

5. Отправка POST запроса

Отправляем данные на сервер. POST используется для отправки данных. Content-Length — количество символов.

POST /task2/index.php HTTP/1.1
Host: u82300.kubsu-dev.ru
Content-Type: application/x-www-form-urlencoded
Content-Length: 13

comment=hello

Ответ сервера: 200 OK

6. Получение первых байт файла

Используем заголовок Range чтобы получить часть файла.

GET /task2/files/file.tar.gz HTTP/1.1
Host: u82300.kubsu-dev.ru
Range: bytes=0-99

7. Определение кодировки index.php

Используем HEAD чтобы узнать кодировку страницы.

HEAD /task2/index.php HTTP/1.1
Host: u82300.kubsu-dev.ru

Кодировка: UTF-8
